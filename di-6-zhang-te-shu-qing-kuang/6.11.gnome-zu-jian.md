# 6.11.GNOME 组件

**表17. GNOME 宏程序组件**

|    Component    |                             描述                             |
| :-------------: | :----------------------------------------------------------: |
|  `gnomeprefix`  |              为 `configure` 提供一些默认位置。               |
|   `intlhack`    | 与 intltool 相同，但要确保使用 **share/locale/** 的补丁。请只在 `intltool` 单独使用不够的时候使用。 |
| `referencehack` |     这个宏是为了帮助将 API 或参考文献分割成自己的端口。      |
