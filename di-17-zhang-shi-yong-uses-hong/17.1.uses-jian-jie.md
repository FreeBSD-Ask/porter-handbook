# 17.1.USES 简介

`USES` 宏使得声明一个 port 的需求和设置变得很容易。他们可以通过选择简单的预设值来增加依赖关系、 改变联编行为、 为包添加元数据， 等等。

本章的每一节都描述了 `USES` 的一个可能的值， 以及它可能的参数。参数在冒号（`:`）之后被附加到值中。多个参数用逗号（`,`）分开。

>**例子1. 使用多个值**
>
> ```shell-sessionl
> USES=	bison perl
> ```

>**例子2. 添加一个参数**
>
> ```shell-sessionl
> USES=	tar:xz
> ```

>**例子3. 添加多个参数**
>
> ```shell-sessionl
> USES=	drupal:7,theme
> ```

>**例子4. 混合在一起**
>
> ```shell-sessionl
> USES=	pgsql:9.3+ cpe python:2.7,build
> ```
